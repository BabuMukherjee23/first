Git commands

1. Track Single file
  -> git add [filename]
  -> git add abc.txt (EXAMPLE)

2. Track multiple file
  -> git add [filename] [another filename]
  -> git add xyz.txt index.html (EXAMPLE)

3. Track all files
  -> git add .

4. Commit changes
  -> git commit -m "YOUR MESSAGE"
  -> git commit -m "This is first commit"(EXAMPLE)

5. Check File status
  -> git status

6. Check branch
  -> git branch

7. Setup origin
  -> git remote add origin "GITHUB REPO LINK" 
  -> git remote add origin https://github.com/xyz/test.git (EXAMPLE)

8. Push code to github
  -> git push origin [branch] 
  -> git push origin master(EXAMPLE)

9. Clone project from Github to Local
  -> git clone "GITHUB REPO LINK"
  -> git clone https://github.com/xyz/test.git (EXAMPLE)

10. Create new branch
  -> git branch [branch-name]
  -> git branch test

11. Switch branch
  -> get checkout [branch-name]
  -> get checkout test
